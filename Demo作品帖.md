## 一、Demo 简介

### 这是什么

**GanttLens** 是一个基于浏览器的**可交互施工项目甘特图管理系统**。在传统甘特图基础上做了三件事：

1. **节点下钻** — 点击任务条弹出 4-Tab 抽屉：子任务 / 文档 / 交付物 / AI 笔记，把项目节点和真实文档深度绑定
2. **AI 实时联动** — 自然语言改里程碑、调进度、跨项目分析，所有相关任务自动级联
3. **计划 vs 实际** — 每任务双轨显示，虚线是计划、实色是实际进度、配合今天线一眼看出延期

形态是 **Web SPA**（单页应用），桌面浏览器打开即用，零安装。

### 面向谁

**核心用户**：弱电 / 建筑 / 装修等行业的**中型项目管理者**，同时管 3-5 个并行项目、需要频繁跟客户 / 供应商对齐进度的角色。

**真实痛点**（我自己干了 5 年弱电设计提炼的）：
- Excel 改甘特图日期要手动算 3-5 个子任务的平移
- 项目文件夹里 100 多个文档，不知道哪个对应哪个任务
- 多个项目并行时，资源冲突 / 延期全靠脑子记
- 周报 / 风险报告全靠人工整理，汇报成本高

### 主要功能

**① 总览大甘特**（首页）
- 一张 Gantt 展示 3 个项目按实际时间平铺
- 三色阶段带（蓝 = 前期设计 / 黄 = 中期施工 / 绿 = 后期验收）
- 两个可拖动里程碑 ◆ 调整阶段边界
- 红色今天线 + 计划/实际双轨对比
- 跨项目冲突自动标红

**② 节点下钻 4-Tab 抽屉**
- 点击任务条 → 右侧抽屉滑出
- Tab 1 子任务：WBS 分解 + 进度条 + checkbox
- Tab 2 文档：关联文件列表（图压井道图 / BOQ 清单 / 合同）
- Tab 3 交付物：验收标准 + 勾选状态
- Tab 4 AI 笔记：摘要 + 风险 + 变更时间线

**③ AI 工作流**（核心差异化）
- 应用内嵌 LLM 聊天
- 「把弱电管线延后 3 天」→ 自动算 → 甘特图 + M2 + 所有关联任务自动顺延
- 7 家 Provider 通用接入（OpenAI / DeepSeek / Moonshot / Zhipu / MiniMax / MiMo / Custom）
- 默认 Mock 模式，零 API 成本演示

## 二、Demo 创作思路

### 灵感来源

我做了 5 年弱电智能化设计（博物馆 / 数据中心 / 办公室装修 / 海外安防），最熟悉的痛苦就是：项目多到飞起来，但管理工具全是过时的。Excel 甘特图、OneNote 笔记、邮件、微信群、纸质 checklist... 5 套系统并行，每次改一次进度要发 3 遍消息（团队 / 客户 / 自己备忘）。

我就在想：能不能让 AI 帮我处理这些机械工作？比如「把 M1 开工延后 3 天」直接说人话，AI 算清楚所有联动，我只在最后审一眼？

### 评估过的 3 个方案

**方案 A：纯静态 HTML**
- ✅ 零依赖、起动快
- ❌ 不能拖拽、不能联动

**方案 B：React + 真实 LLM**
- ✅ 体验真实
- ❌ CORS、密钥泄露、API 成本、演示稳定性

**方案 C：React + Mock LLM 优先 + 真实 LLM 可选** ⬅️ **选这个**
- ✅ 演示 100% 稳定（Mock 不依赖网络）
- ✅ 评审能看 2 种 AI（预设响应可控 + 真实响应可惊艳）
- ✅ Key 用户自填 → 存 localStorage → 绝不入 Git
- ✅ 7 家通用 LLM 协议（OpenAI 兼容），任何大模型都能接

### 为什么做这个方向

核心判断：**TRAE 评审维度（创新 / 完成度 / 体验 / 文档）** 4 项都拿得下：
- 💡 **创新性**：节点下钻 + AI 联动 + 里程碑平移算法，MS Project、飞书项目、Trello 都没这套
- 🔧 **完成度**：总览 + 详情 + 4-Tab 抽屉 + AI 聊天全跑通，不是概念稿
- 🎯 **体验感**：Hover 预览、拖拽改里程碑、抽屉滑入、Mock AI 响应快到无感
- 📄 **文档质量**：TRAE 开发过程全记录 + 6+ 截图 + 4 Session ID

## 四、技术栈

```
frontend:  React 18 + TypeScript + Vite
state:     Zustand (3 stores)
styling:   Tailwind CSS
gantt:     frappe-gantt + 自定义 SVG 覆盖层
ai:        通用 OpenAI 兼容协议 (7 Provider)
data:      seed JSON + localStorage
deploy:    Vercel
```

总计 **~40 个文件 / 约 3000 行代码**，1 人 7 天完成。
